What is the project about?

ABMVis3000 is an urban traffic viewer, which displays agents and vehicles moving in a transport network. The application allows you to view data from simulations or real observations.

Why the project?

To work with urban traffic data, it is very beneficial to use a graphical viewer. However, few applications allow you to simply view this data while remaining accessible. The goal of ABMVis3000 is to allow clear and easy-to-configure traffic data visualization while being free and cross-platform.

What we offer

For The Developer

modular software

functional parser for the output data of MATSim, one of the main urban traffic simulation software

possibility of implementing a custom parser, suitable for any type of data

ability to implement a custom data model to support new types of objects

For The User

a simple and useful visualization tool

visualisation of urban trafic: allows better understanding of data and doing research or forcasting work

application of graphic filters: allows to personalize the display and thus obtain a more relevant and readable visualization

statistical analysis: allows to study the data from a mathematical angle thanks to curves and statistical values
